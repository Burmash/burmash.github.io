Моделирование статической модели ноутбука
============================================

Введение
----------

Настоящее техническое задание определяет требования на моделирование статической 3D модели ноутбука
для использования в коммерческих целях АНПОО "Академическая школа Информационных Технологий"

Основание для создания модели
--------------------------------

Основанием для создания модели является моделирование статической модели
с использованием референсов для коммерческого использования и продвижения проекта

Назначение модели
------------------

Модель предназначена для коммерческого использования и продвижения проекта

* Использование модели в качестве игрового ассета;

* Использование модели в маркетинговых материалах (рекламные ролики, плакаты, презентационные материалы);

* На выставках и форумах, в которых принимает участие Заказчик;

* На официальных и публичных мероприятиях, организуемых Управляющей компанией;

 В ходе переговоров с потенциальными инвесторами и Партнерами Управляющей компании.

Требования для создаваемой модели
-----------------------------------


#. Срок изготовления модели – не более 6 рабочих дней;

#. Создание трехмерной хайполи модели с размерами: высота 10 см, ширина 35 см, глубина 24 см;

#. Создание трехмерной лоуполи модели с использованием сглаживания и размерами: высота 10 см, ширина 35 см, глубина 24 см;

#. Максимально допустимое кол-во треугольников не должно превышать 30 тысяч;

#. Создание UV-развертки для лоуполи модели;

#. Модель должна быть симметричной;

#. Модель должна быть направлена лицом по оси Y+;

#. Текстуры для модели должны быть в формате .png иметь такие карты как:

        a. Normal Map

        b. Base Color

        c. Height Map

        d. Metallic Map

        e. Roughness Map

Требование к качеству
----------------------


#. На модели не должно быть артефактов;
#. Шейдинг модели должен быть гладким и без треугольных форм;

Условия эксплуатации
----------------------


Использование модели допускается только заказчиком.

Требования к ПО
-------------------


#. Модель должна быть создана в таких программах как Maya или Blender.

#. Текстуры для модели должны быть созданы в такой программах как Substance Painter или Adobe Photoshop.

#. UV-развертка должна быть создана в такое программе как Marmoset Toolbag 3.

Требование к программной совместимости
-------------------------------------------

#. Модель должна быть в формате .fbx

#. Модель должна поддерживать работу в среде разработки Unity.

Требования к дизайну
---------------------


#. Цвет клавиш: #000000 (RGB: 0, 0, 0)

#. Цвет корпуса: #808080 (RGB: 128, 128, 128)

#. Цвет экрана: #000080 (RGB: 0, 0, 128)

#. Цифры на клавишах арабские

#. Алфавит на клавишах русский и английский

Этапы моделирования
------------------------


Работа по созданию данной модели разделяется на следующие этапы:

#. Изучение предоставленных референсов. (1 час)

#. Создание хайполи модели. (1-2 дня)

#. Создание лоуполи модели. (1 день)

#. Развертка UV-карты. (1 час)

#. Запечка карт. (1 час)

#. Наложение текстур. (1-2 дня)

#. Отправка модели (по завершению работы)

Порядок контроля и приёмки
---------------------------


Должна быть произведена проверка модели на наличие артефактов и несоответствий с предоставленными референсами.
